# ts-template

自分用のTypeScriptテンプレート．

## setup

```
yarn
```

## build and run

```
yarn build
yarn start
# for debugging
yarn debug
```

## linting

```
yarn lint
# lint and fix
yarn lint:fix
```

## test

```
yarn test
# for watching
yarn test:watch
# for debugging
yarn test:debug
```

## clean artifacts

```
yarn clean
```

## notes

* eslintの`brace-style`のような細かいルールがないので，prettierは使わないことにした．
